Use this tool to induce a numeral grammar in any natural language or natural-like conlang with a minimum number of inputs. 

Make sure that you have installed Python and the packages from 'requirements.txt'.

Start the app with 'python numeral_learner_app.py'

Click on the the buttons and fill them with numeral words. 
The underlying reinforcement learning algorithm eventually auto-fills many buttons which are then shown in lighter green or yellow.
The auto-filled fields (especially the yellow ones) can be wrong, so, review them carefully.
Click on incorrect words to correct them.

Use the test area in order to review how the learned grammar translates numbers to words and vice versa.
